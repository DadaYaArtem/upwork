# Concierge service / Perfect Live

## Общая информация
- Клиент: Concierge service / Perfect Live
- Локация: Estonia
- Отрасль: Concierge service
- Формат: NFT-based loyalty program

## Проблема клиента
Loyalty program functionality for the existing platform Perfect Live.

## Решение
Web application + Blockchain

## Ключевые фичи
- Loyalty program based on NFT. Users collect points to reach a certain level (1-5) of the loyalty program. By reaching a new level
- users are allowed to buy an NFT. Each NFT provides users with more benefits
- discounts
- or exclusive deals.
