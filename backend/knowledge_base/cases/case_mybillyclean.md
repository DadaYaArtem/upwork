## Проблема клиента
Build a Shopify based web e-commerce app

## Решение
E-commerce web landing page for the Soap product. User will be able to order a product to be delivered to his home. User will be able to buy a subscription to get the soap each certain period of time.

Задачи команды: Shopify

## Ключевые фичи
- The product itself is a soap that does not require water to be used. A user can just simply pull the soap and rub it into their hands until the hands are dry. So it's kind of a cream structure soap that does not require water to be used. And this soap is not as unhealthy for the skin as a hand sanitizer.
