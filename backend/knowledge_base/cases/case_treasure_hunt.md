- A core app rebuild including a Web3‑native layer, staking and reward mechanics, first on‑chain quests, plus Social‑Fi framework launch and new game modes (PvE/PvP, custom skins), global scaling efforts with DAO‑like governance, and API access for partners were all outlined as upcoming — likely slated for 2025‑2026 ([treasurehunters.io]).
