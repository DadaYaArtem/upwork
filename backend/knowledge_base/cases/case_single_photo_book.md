# Single Photo Book

## Общая информация
- Клиент: Single Photo Book
- Локация: South Africa, Cape town
- Отрасль: Social

## Проблема клиента
Комфортная апка для сохранения фотографий

## Решение
The product Single Photo Book is a cross-platform application that allows users to create a photobook from their mobile phones and order it to selected locations. The application provides an opportunity to customize the book however the user wants.

Задачи команды: Cross-platform app, ReacTNative, PHP

## Ключевые фичи
- The main goal of the product is to help users to create and purchase a photo book whenever they want and wherever they are. The Single Photo Book system will provide users with the ability to place images on each page exactly as they want
- add text under each of the photos
- and edit the font / color of the text. The application will provide users an easy and understandable way for creating the photobooks.
